# 종이 접기 3D

실물 종이를 아끼면서 접기 퍼즐을 실험할 수 있는 웹 도구.

**사용:** https://anjaeheung.github.io/paper-fold/

- 🖼 앞면/뒷면 이미지 업로드 (브라우저 안에서만 처리, 서버 전송 없음)
- 🤏 접기: 잡아서 끌면 실시간으로 접힘 · 접힌 부분을 반대로 끌면 펴짐 (초록 선)
- ✂ 오리기: 선/네모로 자르기 (네모는 영역만 뚝 떼어냄)
- ✋ 옮기기: 조각 이동 + 15° 회전
- 🔄 뒤집기 · 💡 비추기(앞뒤 투과) · 🗂 겹치기(자료를 반투명하게 포개기)
- 자석 스냅(모서리·중간점·접힌 선), Shift로 해제

## 자료 내장 (선택)

`assets/` 폴더에 파일을 넣으면 자동으로 "정보 N" 탭이 생깁니다:

| 파일 | 의미 |
|---|---|
| `N-front.png` / `N-back.png` | 정보 N의 앞면/뒷면 (N = 1~12) |
| `N-open.png` | 개봉 버튼으로 전환되는 모습 |
| `0-front.png` / `0-back.png` | 키트(봉투 등): 첫 화면이 되고, 개봉하면 자료들이 펼쳐짐 |

빌드 도구 없음 — Three.js CDN을 쓰는 정적 페이지. GitHub Pages로 배포.
